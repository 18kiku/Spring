# SpringMall

쇼핑몰 프로젝트로 공부하는 Spring Framework

개발환경

OS : Windows 11

개발언어 : Java, JSP

Java Version : JavaSE-1.8 (jre1.8.0_321)

IDE : spring-tool-suite-3.9.15.RELEASE-e4.16.0

DB : Oracle 11g

#다음 계획

대사관별 다른 영사 정보를 종합하여 비교할 수 있는 페이지를 만들어보자
